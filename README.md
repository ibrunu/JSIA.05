Requisitos do Projeto:
* Adicionar Tarefas:
 - Solicitar ao usuário o nome de uma tarefa e adicione-a ao final do array usando o método push().

* Listar Tarefas:
 - Usar um loop for ou for...of para percorrer o array de tarefas e exibi-las com seus índices.

* Remover Tarefas:
 - Solicitar ao usuário o índice da tarefa que deseja remover e utilize o método splice() para removê-la do array.

* Concluir Tarefas (Opcional):
- Peça ao usuário o índice de uma tarefa e marque-a como concluída, modificando a string para algo como "✅ Lavar a louça".

* Interação Contínua:
 - Utilizar um loop while para continuar solicitando ações do usuário (adicionar, listar, remover, concluir) até que ele escolha sair.
